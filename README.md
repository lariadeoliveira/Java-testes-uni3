# Unidade 3 – Exercícios em Java

Repositório com as soluções para os **10 exercícios da Unidade 3**, cobrindo:

- Árvore binária e BST  
- Busca linear e binária  
- Grafos (lista de adjacência)  
- BFS e DFS  
- Dijkstra (caminho mínimo ponderado)  
- Ordenação topológica (DAG)

---

## Estrutura

| Exercício | Arquivo | Descrição |
|---|---:|---|
| 1 | `Ex01_BinaryTree.java` | Árvore binária de livros + travessias (pre, in, post) |
| 2 | `Ex02_ValidateBST.java` | Validação se uma árvore é BST |
| 3 | `Ex03_LinearSearch.java` | Busca linear em array |
| 4 | `Ex04_BinarySearch.java` | Busca binária em array ordenado |
| 5 | `Ex05_GraphAdjacency.java` | Representação de grafo (lista de adjacência) |
| 6 | `Ex06_BFSShortestPath.java` | BFS para caminho mais curto em grafo não ponderado |
| 7 | `Ex07_DFS.java` | Busca em profundidade (DFS) |
| 8 | `Ex08_BSTFindMin.java` | Encontrar mínimo em BST |
| 9 | `Ex09_DijkstraShortestPath.java` | Dijkstra para grafos ponderados |
| 10 | `Ex10_TopologicalSort.java` | Ordenação topológica (Kahn) |
